# PlotPilot Pure

PlotPilot 纯前端版本 - 无需后端服务器，所有数据存储在浏览器中。

## 原仓库
https://github.com/shenminglinyi/PlotPilot

## 核心功能

- ✅ Story Bible (人物、地点、世界观)
- ✅ 知识图谱 (自动提取三元组)
- ✅ 伏笔台账 (埋设与回收追踪)
- ✅ 节拍表 (三幕式故事结构)
- ✅ 章节管理 (增删改查)
- ✅ 自动驾驶 (连续生成章节)
- ✅ 风格分析 (简单版本)
- ✅ 数据导出/导入
- ✅ 支持 OpenAI/Claude API

## 一键安装 (Termux)

```bash
curl -O https://raw.githubusercontent.com/Neohero521/PlotPilot-Pure/main/Install.sh && bash Install.sh
```

## 启动

```bash
plotpilot-pure
```

## 技术栈

- Vue 3 + TypeScript
- Naive UI
- LocalStorage/IndexedDB
- Vite
